You can choose to use with yarn or npm.

## Install
Install global and local dependencies<br>

### Yarn
```sh
yarn
```

### NPM
```sh
npm i
```

## Running locally

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Yarn
```sh
yarn dev
```

### NPM
```sh
npm run dev
```
